# AnalystLab-Africa-Internship-Program

## Overview
---
Welcome to my AnalystLab Africa Internship Portfolio repository. This repository showcases the projects, assignments, case studies, and hands-on exercises I completed during my 8-week Data Analytics Internship at AnalystLab Africa, held from **1st June 2026 to 1st August 2026**.

Over the course of the internship, I developed practical experience in data analytics, business intelligence, data visualization, SQL, Python programming, and ETL (Extract, Transform, Load) pipeline development. Each project highlights my ability to collect, extract, clean, transform, analyze, visualize, and interpret data to generate actionable insights that support informed, data-driven decision-making.

Analyze datasets to identify patterns, trends, anomalies, and insights using descriptive statistics and visual exploration techniques.

### Key Activities

• Data Cleaning • Data Preparation • Handling Missing Values • Exploratory Data Analysis • Statistical Summaries • Data Visualization

### Skills/Tools Applied

• Excel • SQL • Data Interpretation

## Week 3: SQL and Data Querying

### Objectives

Extract, manipulate, and analyze data stored in relational databases using SQL.

### Skills/Tools Applied

• SQL • Database Management • Query Optimization
